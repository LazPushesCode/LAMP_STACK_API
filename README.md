# COLORS web application

# Description
The COLORS web application is a COP 4331C assignment part of our lab.
The application allows users to interact with the website using front-end technologies.
It allows users to login and manage colors on a database.

## Technologies
  - Frontend: HTML, JavaScript, CSS
  - Backend: PHP
  - Database: MySQL
  - Server Stack: LAMP(Linux, apache, MySQL, PHP)
  - Version Control: Git, GitHub

## Setup Instructions
1. Clone this repository
   - git clone https://github.com/LazPushesCode/LAMP_STACK_AP
2. Place inside your own web directory (/var/www/html)
3. Setup the database:
   - Create a MySQL database
   - Create user
4. Startup Apache and MySQL
5. Go to application in your browser:
   - http://domain_or_IPAddress

## Assumptions and Limitations
  - The application assumes the user already has a LAMP environment set up.
  - User must manually insert logins directly into MySQL database.
  - ChatGPT used as guidance for command line prompts and file management.
